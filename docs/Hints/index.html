    Notes

    • Below is a full list of hints as copied from html/locale_en.json
    • Best way to submit additions/changes is to create PR with changes to html/locale_en.json
      Alternatively, you can use discussions or issue tracker to suggest changes and they will be applied manually
    • Add or edit hints displayed for each UI item
    • Do not change label directly
      if you have a suggestion for a better naming, place it in localized field so changes can be tracked
    • If you want to validate you JSON file before submitting,
      save it locally and run python cli/validate-locale.py
    • If you want to add additional hints, just follow the template
      and create new entry where label field matches exactly what is visible in the UI
